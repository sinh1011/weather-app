# 🌤️ Weather App

Ứng dụng xem thời tiết hiện tại theo thành phố, giao diện đẹp và dễ sử dụng.

## 🚀 Demo

**🔗 [Xem live tại đây](https://sinh1011.github.io/weather-app/)**

## ✨ Tính năng

- Tìm kiếm thời tiết theo tên thành phố
- Hiển thị nhiệt độ, độ ẩm, tốc độ gió
- Icon thời tiết trực quan
- Giao diện responsive, hiện đại
- Hỗ trợ bàn phím (Enter để tìm kiếm)

## 🛠️ Công nghệ

- HTML5
- CSS3 (Flexbox, Animations)
- JavaScript (ES6+, Fetch API)

